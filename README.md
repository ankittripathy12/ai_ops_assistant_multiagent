üìÅ Project Structure
=====================
The project is organized into the following folders:

* `agents`: Contains the Planner, Executor, and Verifier agents, which are responsible for automating tasks and making decisions.
* `llm`: Contains the LLM client and utilities, which provide natural language processing capabilities.
* `tools`: Contains the GitHub Tool, Weather Tool, and Base Tool, which provide integrations with external APIs and services.
* `config`: Contains configuration files and utilities, which manage environment variables and settings.
* `app`: Contains the Streamlit web interface, which provides a user-friendly interface for interacting with the platform.
* `main`: Contains the FastAPI app, which provides a RESTful API for interacting with the platform.
